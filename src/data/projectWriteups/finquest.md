FinQuest is a financial education platform for digitally native Gen Zers to learn about investing and personal finance to build long-term wealth targeted to their goals.

It provides a portfolio management system with realtime financial data, AI-powered learning paths with interactive lessons and quizzes, along with gamified systems to incentivize repeated learning and engagement.

## Tech stack

### Frontend

-   **NextJS** with Typescript and Mantine UI

### Backend

-   **FastAPI** with Python
-   **PostgreSQL** for storing user profiles and portfolio data
-   **SQLAlchemy** for database interactions

### Resources

-   **Yahoo Finance API** for realtime financial data
-   **Google Gemini API** for AI-powered learning paths and quizzes
